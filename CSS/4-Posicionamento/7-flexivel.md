# Flexbox
  * Permite posicionar os elementos dentro da caixa
  * Controle horizontal ou vertical
  * Alinhamento, direcionamento, ordernar e tamanhos

<style>
  div.parent {
    display: flex;
  }
</style>

## flex-direction
  * Qual a direção do flex: horizontal ou vertical
  * row | column

## alinhamento
  * justify-content
  * align-items
