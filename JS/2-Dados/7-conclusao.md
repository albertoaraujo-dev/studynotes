# Tipos de dados

Conforme o ECMAScript standard temos 9 tipos de dados:

* Data types
  * Primitive / Primitive value
  * Structural
  * Structural Primitive

## Primitivos (não é um objeto, são imutáveis)

* String
* Number
* Boolean
* undefined
* Symbol
* BigInt

## Estruturais

* Object
  * Array
  * Map
  * Set
  * Date
  * ...
  
* Function

## Primitivos Estrutural / Structural Root Primitive

* null
