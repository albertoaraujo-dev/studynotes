# Objeto
  * tipo de dado estrutural
  * propriedades / atributos
  * funcionalidades / métodos

## Exemplo
  * objeto vazio
    <script>
      console.log({})
    </script>
    
  * objeto com conteúdo
    <script>
      console.log({
        name: "João",
        age: 29,
        andar: function() {
          console.log("andar")
        }
      })
    </script>
